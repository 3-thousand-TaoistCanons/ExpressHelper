# ExpressHelper
## 快递助手（别称：水表助手）

A Material Design App which can help you check the latest status of express package.

一枚帮助你查询最新物流状态的 Material Design 应用。

快递信息API由'爱查快递'提供，建议自行到 <http://api.ickd.cn/users> 申请Token以便调试，目前Token限制每天调用2000次。
